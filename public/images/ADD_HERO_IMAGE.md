# Add Your Hero Image Here

## Instructions:

1. **Save your hero image** in this folder (`public/images/`) with one of these names:
   - `hero.jpg`
   - `hero.png`
   - Or any name you prefer (just update the config)

2. **Update the config** in `src/config/constants.ts`:
   ```typescript
   heroImage: '/images/your-image-name.jpg',
   ```

## Image Requirements:

- **Format**: JPG, PNG, or WebP
- **Recommended size**: 800x800 pixels (square) for best results
- **File size**: Under 1MB (compress if needed)

The image will appear as a circular profile photo in the hero section!
