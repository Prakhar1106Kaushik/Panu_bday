# Add Your Aurora Background Image Here

## Instructions:

1. **Save your aurora image** in this folder (`public/images/`) with the name:
   ```
   aurora-background.jpg
   ```

2. The image will automatically be used as the hero background!

## Image Requirements:

- **Format**: JPG, PNG, or WebP
- **Recommended size**: 1920x1080 pixels or larger
- **File size**: Under 2MB (compress if needed for better performance)

## Alternative: Use Image URL

If you prefer to use an image URL instead, edit `src/config/constants.ts` and change:
```typescript
auroraBackground: 'https://your-image-url-here.jpg',
```
